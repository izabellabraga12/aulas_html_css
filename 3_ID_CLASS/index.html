ID e CLASS

O ID funciona como um CPF. Assim como só posso ter um nuúmero de CPF, cada elemento só pode ter um valor de ID.O CPF é unico , ou seja, ningém pode ter um CPF igual ao meu.Isso signfica que o ID também é unico, então não é possivel repetir o mesmo valor duas vezes.

CLASS

O CLASS funciona de maneira oposta ao ID em que você só pode ter um valor no CLASS você pode ter varios valores. Também diferente do ID, em que não se pode repetir um valor, no cLASS você opode utilizar o mesmo valor para varios elementos.
